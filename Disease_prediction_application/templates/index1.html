About the Intelligent System:
Accuracy

We have used over four machine learning algorithms that enhance the accuracy of the model, upto 90%.

Portability

This system can be accessed anytime, anywhere reflecting quick results.

Objective

The main objective of this system is to predict the disease based on a list of symptoms.

Algorithms Used:

K-Nearest-Neighbour


We have used K-Nearest-Neighbour to classify our dataset and achieved ~92% accuracy.

Decision Tree


Decision tree is the first prediction method we have used in our project. It gives us an accuracy of~95%.

Random Forest


we have used random forest classifier with 100 random samples and the result from this algorithm is ~95% accuracy.

Naive-Bayes


Through, Naïve bayes algorithm we achieved ~95% accurate predictions.
